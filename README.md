# Let's Go Fox 

방향키를 조작하여 점프를 하여 아이템을 획득하고 몬스터를 없애는 게임

**구현한 기능**

방향키는 A(좌),D(우),SPACE(점프)

맵 아래로 떨어지거나 몬스터와 부딪히면 체력(HP)이 깎임

몬스터 위에 점프하면 몬스터를 잡을 수 있음

아이템 먹을 때마다 점수 획득

팻말 도착 시 다음 스테이지로 넘어감

**개발환경**

Unity 2021.3.28f1

Visual Studio 2022
